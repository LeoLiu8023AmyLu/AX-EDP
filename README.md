# AX-EDP
安夏电子席卡 电子题词机版本 使用Android Studio 编写</br>

## 说明: </br>

* 项目采用 **小米平板2** 作为蓝本开发，分辨率为 ** 2048*1536 ** </br>
* 项目Android 版本使用 **25** 版本 **Android 7.1.1 (Nougat)** </br>
